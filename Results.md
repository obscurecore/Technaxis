## REST API endpoints :
```text
HTTP METHOD   URI           Description    

GET           /book{id}     Find a book    
GET           /books{page}  Get list of books    
POST 	      /books 	    Save a book.    
PUT 	      /books/{id} 	Update a book or save it.
PATCH 	      /books/{id} 	Update a single field (readAlready)


```
